Use Strong Passwords

Never reuse passwords

Use strong passwords
just as important, Never reuse passwords
If you reuse a username and password, then if one site is hacked and their database of user accounts is stolen, then whoever has that information can immediately log in as you on any other website where you used the same password.
https://haveibeenpwned.com currently tracks 187 hacked websites totalling over 2 billion usernames and passwords (myspace, linkedin, adobe...)

Password Manager

1Password, KeePassX, LastPass, Dashlane

These are just a few password managers with good reputations for security and usability.
I personally use and recommend 1password, but there are factors to be considered for all of these... we can talk after.
You can generate and securely store a different strong password for every account.
All of these are encrypted and locked behind one secure master password.
There are browser extensions to easily enter a password in a browser.
And there are mobile apps whioch integrate with moany of the apps you use.

Use Two-Factor Authentication

Something you know

Something you have

Two-factor authentication is a system that adds a second factor to the process of logging into a website.
The first factor is something you know, your password... but this doesn't change often and can be intercepted and used by someone else. (or you could have reused passwords and had anopther site's database hacked)
The second factor is something you have, which is much more difficult to steal over the internet....
This is usually your smartphone, which can use a special authenticator app tyo generate unique, secure, ever-changing codes to authenticate qith a website.
  • Two-factor authentication
  • Enable it everywhere you can (Use 2FA wherever available)
  • Prefer tokens instead of SMS
  • Your email account isz the master key to your digital life
  • protect it with 2fa
  • protect other account password resets with 2fa
